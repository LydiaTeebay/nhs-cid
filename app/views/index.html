NHS login Prototype

For use with onboarding partners

Instructions

  • A connected service can pass to the NHS login start page in prototype at the following URL:
    http://nhslogin-demo.herokuapp.com/create-account/demo/start
  • To simulate NHS login passing the user back to a connected service after a successfull login a 'returnUrl' parameter can be set in the URL.
  • On certain screen we display the name of the connected service. This can also be set in the 'serviceName' parameter.
  • To ensure the most recent protype changes are visible ensure the parameter 'service' is set to 'app4'
  • The full url will look like something like this: http://nhslogin-demo.herokuapp.com/create-account/demo/start?service=app4&serviceName=Your%20Health%20Service&returnUrl=https://nhslogin-demo.herokuapp.com/your-service&devMode=undefined
