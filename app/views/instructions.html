NHS login prototype instructions

These instruction help you can plug in your website or app to the NHS login start page

  1. To pass the user back to your website or app after a successful login, set a 'returnUrl' parameter in the url.
  2. On certain screens we display the name of the website or app. Set this in the 'serviceName' parameter.
  3. Make sure the parameter 'service' os set to 'app4'
  4. The full url should look something like this: http://nhslogin-demo.herokuapp.com/create-account/demo/start?service=app4&serviceName=Your%20Health%20Service&returnUrl=https://nhslogin-demo.herokuapp.com/your-service&devMode=undefined
